ls     -    показывает все файлы в папке внутри текущей директории
ls test    -    показывает все файлы и папки внутри папки тест
ls -a     - показывает также скрытые файлы и папки
ls -l     -показывает более подробную информацию

cd     - переходит в домашнюю папку
сd test   - переходит в папку тест
cd .     - переходит в текущюю папку (остается в ней же)
сd ..    - переходит в предыдущюю папку
cd test/test2    - переходит в папку тест2, которая находится в папке тест

mkdir test    -  создает папку тест
mkdir test/test2  -  cоздает папку тест2 в папке тест    
mkdir test1 test 2   -   создает папки тест1 и тест 2 
touch test.txt      -   создает файлы тест.тхт
touch test/test.txt     -   cоздает файл тест.тхт внутри папки тест
touch test1 test 2   -   создает файлы тест1 и тест2

rm test   -     удаляет файл тест если это не папка
rm -rf test    -     удаляет файл или папку со всем содержимым
mr -rf /    -   !внимание не писать!

pwd     -   показывает где мы сейчас находимся

сat test.txt    -    показывает содержимое файла
nano test.txt   -   открывает тест.тхт в редакторе кода нано в терминале 
echo "	-	записывает текст в файл
code test.txt   -   открывает тест.тхт в редакторе кода вскоде
sudo    -    права супер юзера
sudo apt install git    -   установить систему гит
